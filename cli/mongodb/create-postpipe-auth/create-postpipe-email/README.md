## Usage

Run this command in your Next.js project root:

```bash
npx create-postpipe-email
```

## What it does

1.  Installs `resend` dependency.
2.  Creates `src/lib/sendEmail.ts` (or `dlib/email/sendEmail.ts`) utility.
3.  Configures types for email payloads.
